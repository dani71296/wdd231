MotorCycle Hub - Site Plan

Site Name

MotorCycle Hub

Selected because it clearly represents a hub for motorcycle resale and repair services, making it easy to remember for potential customers.

Optional domain: motorcyclehub.com

Site Purpose

This website provides information and services for motorcycle enthusiasts, including:

  • Motorcycle resale listings with photos and prices.
  • Repair services and maintenance options.
  • Contact information for easy communication.

Scenarios

  • Where can I find motorcycles available for resale in my area?
  • How do I book a repair or maintenance service for my motorcycle?

Color Schema

  • Dark Blue (#1E3A8A): Headings and important text.
  • Amber (#FBBF24): Links, buttons, accents.
  • Light Gray (#F3F4F6): Background color for sections.

Typography

  • Arial, sans-serif: Body text and general content.
  • Verdana, sans-serif: Headings for emphasis.
